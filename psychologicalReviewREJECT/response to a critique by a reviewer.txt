Although I use terms that may be conceptually laden to describe the different features in the model, it is not my contention that infants process or see these events in the same way. Instead, it is assumed here that infants process these features as they would any other low-level features that readily can be observed in the perceptual array. Additionally, one may argue that I have simplified the model in ways that are not true for infants. IN particular, by choosing which features to implement, I have chosen for the model which features to focus on. Yet, in the real world, infants must discover which features to attend to. I have two responses, it has been shown elsewhere if some features are more salient than others, models can discover which of the features are causally meaningful, even if the features are perfectly correlated. Second, the features I have chosen here are exactly the kinds that infants attend to over others in the real world. For instance, we know that from birth or shortly there after, infants attend to movement over nonmovement. As such, movement seems like a good candidate feature to include in a model that simulates how infants process actions in the real world.
